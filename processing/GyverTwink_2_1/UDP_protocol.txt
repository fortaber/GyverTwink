Формат пакета: {'G','T',x,x...}, где x - байты [0-255]
Приложение - мастер. Отправляет:

{0} - запрос IP
esp отвечает {0, ip}, ip - последний октет адреса

{1} - запрос настроек
esp отвечает {1, 0, колво_led/100, колво_led%100, питание, яркость, автосмена, случайная_смена, период, таймер активен, время таймера}
и затем {1, 1, pos1x, pos1y, pos2x, pos2x ...} -отправка 60 координат светодиодов
и затем {1, 2, pos61x, pos61y, pos62x, pos62x ...} -отправка след. 60 координат светодиодов
и т. д.

{2, 0, am1, am2} - отправить кол-во ледов (am1 = колво_led/100, am2 = колво_led%100)
{2, 1, val} - отправить состояние питания
{2, 2, val} - отправить яркость
{2, 3, val} - отправить флаг авто смены режимов
{2, 4, val} - отправить флаг случайной смены режимов
{2, 5, val} - отправить период смены режимов
{2, 6} - следующий эффект
{2, 7} - отправить состояние таймера выключения
{2, 8, val} - отправить время до выключения

{3, 0} - запуск калибровки
{3, 1, n/100, n%100, x, y} - включить светодиод под номером n (от 0 до N) и передать координаты по сетке ПРЕДЫДУЩЕГО светодиода
{3, 2} - остановить калибровку
{3, 3} - выкл светодиоды
{3, 4, n/100, n%100} - вкл светодиод под номером n

{4, 0, n} - выбрать пресет под номером n
esp отвечает {4, favorite, масштаб, скорость, эффект, палитра}

{4, 1, val} - установить флаг избранного val
{4, 2, val} - установить масштаб val
{4, 3, val} - установить скорость val
{4, 4, val} - установить эффект val
{4, 5, val} - установить палитру val

{5, 0} - запуск режима рисования
{5, 1, n/100, n%100, h, s, b} - вкл n-й светодиод цветом в HSB формате
{5, 2} - остановка режима рисования
